# My Personal Portfolio 🚀

Welcome to my personal portfolio repository! This is a single-page website built to showcase my projects, skills, and contact information. 

## 🌐 Live Demo
You can view the live site here: https://cpmashir.vercel.app/
## 🛠️ Built With
* **HTML5** - Structure and Content
* **CSS3** - Styling and Layout (Internal)
* **JavaScript** - Interactivity (Internal)
* **Vercel** - Deployment & Hosting

## 📁 Project Structure
The entire site is contained within a single file for maximum performance and simplicity:
* `index.html` - Contains the structure, styles, and logic.

## 🚀 How to Make Changes
Since this project is linked to Vercel via GitHub, updating the site is easy:

1. **Edit the code:** Open `index.html` and make your changes.
2. **Commit and Push:**
   ```bash
   git add index.html
   git commit -m "Describe your changes here"
   git push origin main
